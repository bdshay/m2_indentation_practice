Indentation Practice

  • Due Monday by 11:59pm
  • Points 10
  • Submitting a text entry box or a file upload
  • Available until Mar 29 at 11:59pm

Assignment Objectives:

By the end of this lesson, you will be able to:

  1. Demonstrate your ability to properly indent code

 

You can choose to complete this activity in one of two ways:

  • For a challenge: try completing the activity on your own, then watch the video to check your work and make sure that everything is correct. The video also has additional explanations and tips about the code, so you should watch it for the additional information about the code that should help with your assignments in the course.
  • For step-by-step instructions: complete the activity by working along with me in the video.

 

 

Instructions:

1. Fork a Copy of the m2_indentation_practice Repository

  1. Make sure that you are signed in to your account, and go to the repository (m2_indentation_practice) in your browser.
  2. Fork a copy of the repository into your account (instructions can be found in the document, follow the link to find them)
  3. Clone your new/forked repository to your computer. Be sure to clone this into your m2 folder when choosing the path (instructions can be found in the document, follow the link to find them).

 

2. Open the Repository in Visual Studio Code

  1. From GitHub Desktop, open the repository in Visual Studio Code, then open the index.html in the repository
  2. Review the code structure for the tags learned in this zyBooks module.
  3. Properly indent the code/fix the indentation in the file, and save the changes.
  4. Commit those changes to your repository.

 

3. Push Changes to GitHub

  1. From GitHub Desktop, push your changes to GitHub.
  2. Verify that your m2_indentation_practice repository on GitHub is public and that it contains your updated index.html file.

 

Submission Details:

  • Upload your index.html file to the Canvas assignment link below.
  • Paste a link to your GitHub repository in the submission comments.

Note: If you are unsure how to properly indent code, review the video in zyBooks.
